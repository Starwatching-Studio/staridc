# StarIDC 更新日志

## v1.6.0 (2026-07-19)

### 新功能
- **签到弹窗广告支持独立链接**：每张广告图片可单独设置跳转链接，兼容旧版数据（旧数据自动 fallback 到全局链接）
- **前台广告标识**：所有前台广告位（侧边栏、登录页、签到弹窗、新用户弹窗）均添加小"广告"标签，符合广告法合规要求
- **宝塔面板 API 密钥管理**：域名检查功能改为用户在后台自行配置宝塔面板 API 密钥，不再依赖 MNBT 数据库直连，适配不同部署环境
- **个人中心"我的实例"板块**：在第三方账号绑定上方新增"我的实例"，展示用户所有主机并支持一键跳转管理
- **Session 长期有效**：管理员和用户登录 Session 有效期延长至 30 天（Cookie + 服务端同步），减少频繁登录

### 修复
- **消费统计金额异常**：修复后台消费统计将积分消耗误计为人民币金额的问题，现在充值金额（type=points）与积分消耗（host_buy/host_renew）分开统计
- **工单附件后台不显示**：修复管理员后台无法查看用户上传的工单附件（图片）问题，附件路径自动适配后台目录
- **导出功能点击无下载**：修复数据导出按钮点击后无响应的问题（导出逻辑误放在 POST 分支内，实际为 GET 请求）
- **系统配置页面点击无反应**：修复后台系统配置页面无法加载的问题

### 改进
- **后台侧边栏排序调整**：系统配置移至仪表盘下方，域名检查移至关于项目上方
- **个人中心侧边栏排序**：按使用频率重新排序：个人中心 → 我的主机 → 积分充值 → 工单支持 → 积分账单 → 我的订单
- **升级脚本完善**：upgrade.php 新增调用 `applyDatabaseSchema()` 补齐所有缺失的表/字段/索引，确保旧版用户可无缝升级到 v1.6.0
- **后台广告标识**：后台导航和支付接口处的推荐链接添加"广告"橙色标签
- **易支付广告链接更新**：后台预设广告链接更新为 `http://pay.xzcloud.vip/`
