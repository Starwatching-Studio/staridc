## ✨ 功能特性

### 🖥️ 虚拟主机自动化

- 集成 **MNBT 梦奈宝塔 API**，下单后自动开通虚拟主机
- **多服务器对接**：支持同时对接多个宝塔 MNBT 服务器，不同主机型号可分配到不同服务器开通
- 支持主机**续费**、**暂停/解停**、**删除**、**重置密码**等全生命周期管理
- 多种主机型号配置，灵活设定空间、流量、绑定域名数等参数
- **三级分类**：主机型号支持按一/二/三级分类展示，便于套餐组织
- **时长折扣**：支持月付/季付/半年付/年付/多年付，不同周期可配置独立折扣
- **弹性配置**：同一型号可开启弹性扩展，用户购买时可按需调整空间、流量、域名数

### 💰 积分经济体系

- **签到积分**：用户每日签到随机获得积分（可配置范围），个人中心支持一键快捷签到
- **注册奖励**：新用户注册自动赠送积分
- **推荐奖励**：邀请好友注册获得奖励积分
- **积分充值**：通过易支付在线购买积分
- **自定义充值套餐**：管理员后台灵活添加/编辑/上下架积分充值套餐
- **优惠码**：后台生成折扣优惠码，支持有效期、使用次数、适用型号

### 🔐 用户系统

- 邮箱注册/登录，支持**邮箱验证码**
- **OAuth 聚合登录**：支持 QQ、微信、支付宝、微博、百度、抖音、华为、小米、谷歌、微软、钉钉、飞书、Gitee、GitHub 等第三方登录
- 登录失败锁定机制（5次失败锁定15分钟）
- **"记住我"** 自动登录（30天有效期）
- 找回密码功能
- 邮箱域名白名单限制
- 个人中心支持绑定/解绑第三方账号

### ⚙️ 管理后台

- **仪表盘**：数据统计总览（用户数、主机数、服务器数、订单数、访问量）
- **系统配置**：MNBT 对接、支付接口、邮件服务、OAuth 登录、站点设置
- **服务器管理**：添加/编辑/删除/启停 MNBT 服务器，支持测试连接
- **主机分类管理**：添加/编辑/删除三级分类
- **主机型号管理**：添加/编辑/上下架型号，分配服务器、分类，设置型号级购买限量、时长折扣与弹性配置
- **用户管理**：搜索编辑、批量加积分、批量删除
- **主机管理**：查看/删除/同步
- **充值套餐管理**：自定义积分充值档位，支持上下架
- **优惠码管理**：生成/编辑/启停/删除优惠码
- **工单管理**：查看/回复/关闭/删除用户工单
- **公告管理**：首页公告弹窗
- **消费统计**：订单数据分析
- **域名检查**：通过宝塔面板 API 查询域名站点信息
- **关于项目**：版本检测、赞助入口

### 🎨 主题系统

- 内置多套精美主题：清新薄荷（默认）、新拟态、现代渐变、扁平化风格等
- 一键切换主题，无需修改代码
- 首页采用响应式设计，自动适配各主题配色

### 📧 邮件通知

- SMTP 邮件发送（基于 PHPMailer）
- 注册验证码邮件
- 主机到期前 5 天自动告警（定时任务）
- 工单状态变更通知（管理员回复/关闭时通知用户）

## 🚀 快速开始

### 环境要求

| 环境      | 版本                     |
| ------- | ---------------------- |
| Web 服务器 | Apache / Nginx         |
| PHP     | 7.4+（需 PDO、GD、cURL 扩展） |
| MySQL   | 5.6+ / MariaDB 10.0+   |
| 宝塔面板    | 任意版本（需部署 梦奈宝塔mnbt系统）   |

### 安装步骤

| 步骤 | 操作 |
|:----:|------|
| 1 | **下载源码**，解压到本地 |
| 2 | **上传至服务器**：创建网站（建议 PHP 8.0），将项目文件上传至 Web 根目录 |
| 3 | **运行安装向导**：浏览器访问 `http://你的域名/install/`，填写数据库信息和管理员账号，系统自动建表并写入初始数据 |
| 4 | **配置宝塔 MNBT 插件**（见下方详细说明） |
| 5 | **配置支付接口**：注册易支付商户 → 后台 → 系统配置 → 支付配置 填入商户信息 |
| 6 | **配置邮件服务（可选）**：后台 → 系统配置 → 邮件配置 填入 SMTP 信息 |
| 7 | **配置定时任务（可选）**：设置 Crontab 每天凌晨执行到期告警 |

#### 步骤 4 详细说明：配置宝塔 MNBT 插件

1. 在宝塔面板中安装 **MNBT 系统**插件
2. 登录 MNBT 后台，获取以下三项参数：

   | 参数 | 获取路径 |
   |------|----------|
   | API 地址 | MNBT 后台 → 系统管理 → API 设置（格式：`mnbt系统连接/api/api.php`） |
   | 宝塔编号 | MNBT 后台 → 宝塔列表 → 宝塔编号 |
   | 宝塔调用密钥 | 同上位置（注意：是**宝塔调用密钥**，不是宝塔密钥） |

3. 进入 StarIDC 管理后台 → 系统配置 → MNBT 对接，填入以上三项（此处为默认服务器）
4. 如需对接多台服务器，进入 **服务器管理** 页面，按同样方式添加其他 MNBT 面板即可

> **注意**：API 地址需确保能被服务器直接访问，不要使用经过 Cloudflare 代理的域名

### 默认主机型号

安装向导预设 4 种主机型号，可根据需求自由调整：

| 型号     | 网页空间    | 数据库空间   | 月流量    | 域名绑定 | 价格（积分） |
| ------ | ------- | ------- | ------ | ---- |:------:|
| 🪐 入门型 | 500 MB  | 100 MB  | 10 GB  | 3 个  | 300    |
| 🌙 标准型 | 1000 MB | 300 MB  | 30 GB  | 5 个  | 600    |
| ☀️ 高级型 | 2000 MB | 500 MB  | 50 GB  | 10 个 | 1200   |
| 💎 旗舰型 | 5000 MB | 1000 MB | 100 GB | 20 个 | 2500   |

## ⚙️ 配置指南

### 系统配置

管理后台 `?page=config` 提供完整的可视化配置界面，所有配置项存储在 `config` 表中。

#### MNBT API 配置

系统支持对接多个 MNBT 服务器，在管理后台 → 服务器管理 中添加：

| 配置项              | 说明                 |
| ---------------- | ------------------ |
| 服务器名称            | 自定义标识（如"香港节点"）     |
| MNBT API 地址      | 宝塔面板地址 + MNBT 插件端口 |
| 宝塔编号             | 宝塔面板 API 编号        |
| API 秘钥 (mn_key)  | MNBT 插件通信密钥        |
| 宝塔调用秘钥 (mn_keye) | 宝塔调用密钥             |
| 插件版本 (mn_vs)     | MNBT 插件版本号         |

> 添加服务器后，在主机型号管理中为每个型号分配对应的服务器即可。

#### 宝塔面板配置（域名检查功能）

域名检查功能通过宝塔面板 API 获取站点信息，需在后台 → 系统配置 → 宝塔面板 中添加：

| 配置项 | 说明 |
| ------ | ---- |
| 面板地址 | 宝塔面板访问地址 |
| API 密钥 | 登录宝塔面板 → 面板设置 → API 接口 → 获取密钥 |

> 获取密钥后，需将服务器 IP 加入宝塔面板的 API 白名单。

#### 支付接口配置

| 配置项    | 说明      |
| ------ | ------- |
| 支付接口地址 | 易支付网关地址 |
| 商户ID   | 易支付商户号  |
| 商户密钥   | 易支付通信密钥 |

#### 邮件服务配置

| 配置项      | 说明                     |
| -------- | ---------------------- |
| SMTP 服务器 | 邮件发送服务器地址              |
| SMTP 端口  | 一般为 465（SSL）或 587（TLS） |
| 邮箱账号/密码  | SMTP 认证信息              |
| 发件人地址    | 与邮箱账号一致                |
| 邮箱白名单    | 仅允许指定后缀邮箱注册（留空则不限制）    |

### 积分奖励配置

在管理后台 → 价格设置 中配置：

| 配置项    | 默认值 | 说明           |
| ------ |:---:| ------------ |
| 签到最小积分 | 50  | 每日签到随机积分下限   |
| 签到最大积分 | 100 | 每日签到随机积分上限   |
| 注册赠送积分 | 100 | 新用户注册赠送积分    |
| 推荐奖励积分 | 50  | 邀请好友注册成功奖励积分 |

### 购买限制配置

在管理后台 → 价格设置 中配置：

| 配置项 | 默认值 | 说明 |
| ----- |:---:| ---- |
| 每用户最多购买主机数 | 5 | 全局限制：每个用户可购买的主机总数（0=不限） |

各型号还可在 **主机型号** 页面单独设置限量，实现更细粒度的控制。若型号自身未设置限量，则只受全局限制约束。

### 充值套餐配置

在管理后台 → 充值套餐 中自定义积分充值档位：

| 配置项 | 说明 |
| ----- | ---- |
| 积分数量 | 用户购买后获得的积分 |
| 价格 | 对应人民币金额（元） |
| 排序 | 套餐显示顺序 |
| 状态 | 上架/下架控制 |

### OAuth 聚合登录配置

在管理后台 → 系统配置 → 聚合登录 中启用并配置：

| 配置项 | 说明 |
| ----- | ---- |
| 启用 OAuth | 总开关 |
| 接口地址 | 聚合登录平台地址（如彩虹聚合登录） |
| AppID | 平台分配的 AppID |
| AppKey | 平台分配的 AppKey |
| 启用平台 | 勾选需要开放的登录渠道 |

用户首次通过 OAuth 登录时，若未绑定本地账号，可选择绑定已有账号或自动创建新账号。

### 主机分类配置

在管理后台 → 主机分类 中维护三级分类：

| 配置项 | 说明 |
| ----- | ---- |
| 分类名称 | 前台展示名称 |
| 上级分类 | 一级/二级分类归属，留空为一级分类 |
| 排序 | 同级分类展示顺序 |

添加型号时需指定所属分类，前台 `cart.php` 会按分类层级展示。

### 时长折扣配置

在管理后台 → 主机型号 → 编辑型号 → 时长折扣 中配置：

| 配置项 | 说明 |
| ----- | ---- |
| 月付 | 单月价格（即型号基础价格） |
| 季付/半年付/年付/多年付 | 分别设置对应时长折扣，如 95 折填 `95` |

前台购买时选择时长后，系统按折后价扣减积分。

### 弹性配置

在管理后台 → 主机型号 → 编辑型号 中开启 **弹性配置**：

| 配置项 | 说明 |
| ----- | ---- |
| 最小值/最大值 | 用户可拖动范围 |
| 步长 | 每次调整增量 |
| 单价 | 每增加一个步长需额外支付的积分 |

适用于网页空间、数据库空间、月流量、绑定域名数。开启后用户购买时可按需求弹性扩展。

## 🔧 二次开发

### 主题开发

在 `theme/` 目录下创建新主题文件夹和 `style.css`，覆盖系统定义的 CSS 类名即可。在管理后台 → 系统配置 → 网站设置 中切换主题。

### 核心类扩展

- **MNBT API**：[rd/MNBT\_API.php](rd/MNBT_API.php) — 封装了与宝塔面板的通信，可替换为其他虚拟主机管理接口
- **支付接口**：[rd/PayAPI.php](rd/PayAPI.php) — 易支付对接，可参考实现其他支付通道
- **系统框架**：[rd/bootstrap.php](rd/bootstrap.php) — 核心函数、数据库连接、Mailer 邮件类

### 数据库扩展

所有建表语句在 [install/index.php](install/index.php#L36-L200) 中集中管理，新增字段或表可在该文件中维护。老用户升级请运行 [upgrade.php](upgrade.php)。

### 核心业务流程

```
用户注册 → 获得初始积分
    ↓
每日签到 → 随机获得积分（个人中心一键签到）
    ↓
选购主机 → 积分支付（支持优惠码折扣）
    ↓
根据型号绑定的服务器 → 调用对应 MNBT API 自动开通虚拟主机
    ↓
使用过程中 → 可在个人中心提交工单，管理员后台回复后邮件通知用户
    ↓
到期前 5 天 → 定时任务发送到期告警邮件
    ↓
到期后可续费（消耗积分延长有效期，自动调用对应服务器 API）
```
